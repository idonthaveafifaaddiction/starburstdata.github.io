5.7. Presto on Google Kubernetes Engine#

Network Policy Support#

You need to enable support for Network Policies in Google Kubernetes Engine in order for Presto Network Policies to be effective. For more information, please refer to Enabling GKE network policy enforcement.

Accessing Cloud MySQL or PostgreSQL#

In order to access Google Cloud SQL via private IP, you need to start your Kubernetes Cluster with Alias IP addresses. For more information, please refer to Creating a Cluster with Alias IP addresses.
