Uninstalling Presto
===================
The Presto Application can be deleted from the HDInsight cluster. Microsoft provides documentation for how to `remove HDInsight applications <https://docs.microsoft.com/en-us/azure/hdinsight/hdinsight-apps-install-custom-applications#remove-hdinsight-applications>`_ using the Azure Portal, Azure Powershell, or Azure CLI. 

Additionally, `deleting the entire HDInsight cluster <https://docs.microsoft.com/en-us/azure/hdinsight/hdinsight-delete-cluster>`_ will also remove Presto.
