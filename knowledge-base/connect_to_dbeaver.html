    Connect to DBeaver #

    Overview

    DBeaver is a very popular open-source BI tool used by data analytics teams throughout the world. This article will cover how to configure an unsecured JDBC connection from DBeaver to Presto using the Presto JDBC driver.

    Note: this tutorial was developed on Mac OS, but the steps to configure this should be very similar on any operating system.

    Implementation Please supplement the implementation instructions with the Starburst JDBC documentation.

    First, you will need to download the latest JDBC driver from the Starburst download portal. From here, you can follow this DBeaver tutorial for setting up a JDBC connection. A sample setup is included below to help with a Presto-specific connection.

    Driver Settings Create a new connection in DBeaver and, in the “Select your database” window, select PrestoSQL. Click “Next” and then click the “Edit Driver Settings” button. Select your local Driver .jar file and click the “Find Class” button. Your settings should look similar to this:

    Click the “OK” button to save your changes.

    Note: If you need to change the connection port, the driver settings is where that can be done. For example, if you needed to connect to port 8443 instead of 8080, you would change that setting in the above window.

    Connection Settings Next, you will provide your host, which will be the exposed endpoint of the coordinator or load balancer, as well as a default username. Even though this is an unsecured connection, a username will still be required. You can select “Test Connection …” to see if your connection is successful, and then you can click “Finish” once the necessary details have been populated.

    SSH Tunneling You can set up SSH tunneling, which is likely a requirement for you if trying to connect to a remote cluster. You can set it up as so (you will need to include your passphrase):

    Other Configurations It is possible to utilize SSL, pass in LDAP/Kerberos authentication details, and more. You will need to read about the JDBC driver connection parameters to understand all of the driver’s capabilities.

    Conclusion You should now be able to query your Presto catalogs through DBeaver if you have followed the steps above.
