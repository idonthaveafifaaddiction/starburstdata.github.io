6. Kubernetes#

Starburst Enterprise Presto can be run on Kubernetes clusters as provider for all aspects of the required infrastructure. Kubernetes provides a very powerful and flexible deployment and runtime system.

The following Kubernetes cluster services are supported:

  • Private Kubernetes installations
  • Amazon Elastic Kubernetes Service (EKS)
  • Google Kubernetes Engine (GKE)
  • Microsoft Azure Kubernetes Service (AKS)
  • Red Hat OpenShift
  • Minikube (local testing and development only)

This chapter aims to cover all the above.
