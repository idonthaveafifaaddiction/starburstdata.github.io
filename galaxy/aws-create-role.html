AWS account setup

Create AWS role and configure Starburst’s access to AWS from your AWS console.

AWS prerequisites:

  • Admin access to your organization’s AWS account
  • Permission to create a cross-account role
  • Permission to create an inline policy to a role

Starburst prerequisites:

  • A unique External ID for your Starburst account
  • The account ID for Starburst as an AWS partner
  • An access policy for your new cross-account role (JSON format)
