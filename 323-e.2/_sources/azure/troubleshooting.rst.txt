Troubleshooting
===============
Presto is installed and configured via use of Script Actions. Additionally, you can customize your Presto cluster using Starburst Script Actions or your own. Script action outputs and errors are logged and accessible by the Ambari Web UI and the default storage account with the cluster.

Refer to the Azure HDInsight documentation for more information:

`<https://docs.microsoft.com/en-us/azure/hdinsight/hdinsight-hadoop-customize-cluster-linux#using-the-ambari-web-ui>`_

`<https://docs.microsoft.com/en-us/azure/hdinsight/hdinsight-hadoop-customize-cluster-linux#access-logs-from-the-default-storage-account>`_
