    Tableau #

    Tableau is a popular analytics tool with powerful data visualization capabilities. You can use Tableau to run queries against your Starburst Enterprise platform (SEP) cluster, connecting with the JDBC or ODBC drivers. The recommended connection method is with the JDBC driver.

    You can connect recent releases of Tableau to SEP clusters of version 350-e or older. Newer version are not supported.

    JDBC connection #

    Use the JDBC driver version 350 for maximum compatibility with any SEP clusters version 350-e or lower.

    • Download the recommended JDBC driver for 350
    • The downloaded file is named is presto-jdbc-350.jar.

    Place the file in the Tableau drivers directory, which is:

    • Windows — C:\Program Files\Tableau\Drivers
    • macOS — ~/Library/Tableau/Drivers
    • Linux — /opt/tableau/tableau_driver/jdbc
    Caution: Do not store more than one JDBC driver in this directory.

    You can find more information about Tableau’s JDBC driver usage in the supporting Tableau JDBC documentation.

    With the driver in place, you can start Tableau Desktop. On the main page, select Other Databases (JDBC) and provide the following connection parameters.

    • URL: jdbc:presto://presto.example.com:8080/
    • Dialect: SQL92
    • Username: <username>

    The URL parameter must point to your server hostname and port. Make sure the dialect is set to SQL92.

    Select Sign In to establish the connection. If you are prompted for a password, the server has authentication enabled.

    If you need to set additional connection properties that are not included in Tableau’s connection wizard, you can create a properties file.

    ODBC connection #

    Contact Starburst Support to obtain access to the Starburst ODBC driver. This driver is supported for clusters running SEP release 350 or earlier. The Presto ODBC driver provided by Tableau is not supported.

    Open Tableau and begin the ODBC configuration. On Tableau’s startup page, select Other Databases (ODBC), and configure as follows:

    • Driver: Starburst Presto ODBC Driver
    • Username: <username>
    • String Extras: Driver=Starburst Presto ODBC Driver;Catalog=<catalog>;Host=<host>;Port=<portDriver>;

    The String Extras field supports any of the ODBC connection properties from the Starburst ODBC driver.

    Select Sign In to establish the connection. If you are prompted for a password, the server has authentication enabled.

    The Tableau ODBC documentation contains further information.
