Release 334-e STS (4 June 2020)#

Starburst Enterprise Presto 334-e is the follow up release to the 332-e version.

  • Highlights
  • Notable Breaking Changes
  • Security Changes
  • Connector Changes

The 334-e release includes all improvements from the Presto project releases since Presto 332:

  • Presto 333
  • Presto 334

This release is a short term support (STS) release.

In addition, numerous improvements and additions for enterprise features are added:

Highlights#

  • New Pinot connector
  • New Prometheus connector
  • Improved caching support in Hive connector

Notable Breaking Changes#

  • The required runtime for Presto servers changes from Java 8 to Java 11. More information can be found in Software Requirements.

Security Changes#

  • Add support for using Privacera for system level security with Ranger

Connector Changes#

  • Starburst Teradata and Teradata Parallel Direct connector
    • Add native table operator
    • Fix char mapping
    • Add password path through support
  • Starburst Snowflake connector
    • Fix integer and varbinary mapping
    • Support mixed case column names
  • Starburst Delta Lake connector
    • Avoid populating table statistics into Hive metastore service
    • Map timestamp to Presto timestamp with time zone
    • Add hidden columns with path, file size and file modified time information
