7.11. GCP Marketplace#

Starburst Enterprise Presto is available on the Google Cloud Platform Marketplace. It offers a simple and convenient way to get started with SEP running on the Kubernetes.

The offering is listed in the GCP Marketplace - Kubernetes Applications

The marketplace offering uses the 332-e long term support version of Starburst Enterprise Presto and includes useful documentation to get started.
