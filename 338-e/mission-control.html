4. Mission Control#

Mission Control is a web-based management console for Starburst Enterprise Presto. It allows you to to easily create and manage Presto data sources, the clusters using the data sources and the users allowed to manage everything in Mission Control.

Warning

Mission Control only supports the latest LTS version of Starburst Enterprise Presto.
