2.1. Software Requirements#

The following software requirements apply to running Presto.

Operating System

  • RHEL 6.x, 7.x, 8.x (Red Hat Enterprise Linux)
  • CentOS equivalent to RHEL (Community ENTerprise Operating System)
  • SLES 11 SP4, SLES 12 SP3 (SUSE Linux Enterprise Server)
  • Oracle Linux 7

Hadoop Distributions

  • CDH (Cloudera Distribution Including Apache Hadoop) 5.x
  • HDP (Hortonworks Data Platform) 2.x, 3.1+
  • IBM BigInsights for Apache Hadoop 4.1

Java

  • Java JRE 11 (64-bit)

    • OpenJDK from your Linux distribution
    • Azul Zulu OpenJDK
    • Amazon Corretto
    • Oracle Java

Python

  • Python 2.6.x
  • Python 2.7.x
  • Python 3.x

File Descriptors

The number of open file descriptors and user processes for the presto user should be set to the following values in /etc/security/limits.conf.

presto soft nofile 32768
presto hard nofile 65536

presto soft nproc 32768
presto hard nproc 65536

If you see this error java.lang.OutOfMemoryError: unable to create new native thread then install the JMX connector and run the following query to verify the limit for open file descriptors for your system.

SELECT openfiledescriptorcount, maxfiledescriptorcount FROM jmx.current."java.lang:type=operatingsystem"

Presto Admin

Presto admin usage has its requirements documented separately.
