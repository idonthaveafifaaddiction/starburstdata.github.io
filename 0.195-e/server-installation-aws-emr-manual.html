6. Presto Server Installation on an AWS EMR (Presto Admin and RPMs)

You can use Presto Admin and RPMs to install and deploy Presto on an Amazon Web Services (AWS) EMR.

6.1. Installation on an AWS EMR

Install and manage Presto manually on an AWS EMR in the Amazon cloud. Before installing Presto on an AWS EMR cluster, review the caveats summarized in the following document: Setting up Presto Admin on an Amazon EMR Cluster. Additional administration information can be found here: Presto on AWS EMR.

Download the Presto RPM, presto-server-rpm-0.195-e.0.3.rpm, from https://www.starburstdata.com/download
