What is Starburst Universe?

No matter which Starburst product you use, and what you are trying to achieve, you can find help on Starburst Universe:

  • Installing and operating a Presto cluster on your own infrastructure
  • Using Presto on public cloud provider infrastructure
  • Connecting data sources to query your relational database or your object storage.
  • Using a BI tool to query data with Presto
  • Writing SQL statement to get more insights from your data

Explore the site to find all this and more information in articles, reference documents, videos and others
