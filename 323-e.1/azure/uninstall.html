6.12. Uninstalling Presto

The Presto Application can be deleted from the HDInsight cluster. Microsoft provides documentation for how to remove HDInsight applications using the Azure Portal, Azure Powershell, or Azure CLI.

Additionally, deleting the entire HDInsight cluster will also remove Presto.
